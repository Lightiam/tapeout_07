# Fab Readiness

This folder is the current LightRail AI fab-readiness control point.

Start here:

- `FAB_READINESS_STATUS_2026-07-08.md` - release-gate verdict, blockers, limitations, and next steps.
- `MISSING_DATA_RISK_AND_REQUEST_PROMPTS.md` - why each missing data item matters and paste-ready requests for the NCE, HBM4, TFLN, architecture, RF-front-end, and sourcing owners.

Current status: not ready for production fabrication or assembly release.

The blocking work is pin-complete electrical design data, not Gerber formatting.
The package can move toward final fab release after the authoritative NCE ball
map, HBM4 ball/channel map, TFLN pad map, rail/lane/clock/reset allocation, RF
front-end correction, and BOM/AVL verification are complete.
